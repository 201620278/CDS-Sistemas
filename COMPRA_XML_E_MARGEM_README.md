Ajustes aplicados nesta versão:

- Tela de compras com campo de chave de acesso da NF-e.
- Importação de XML da nota de compra.
- Itens importados continuam totalmente editáveis.
- Margem de lucro padrão de 30% por item.
- Preço de venda sugerido recalculado automaticamente.
- Compra salva atualiza estoque, custo, preço de venda e financeiro.
- Se o produto do XML não existir, o sistema cria um cadastro básico automaticamente.

Observações importantes:
- A importação confiável dos itens está baseada no XML da nota.
- A consulta real da nota diretamente na SEFAZ pela chave não foi ativada nesta versão.
- A chave de acesso é validada e gravada na compra para rastreio.
